## About This Map

Navigating the historic, maze-like corridors of UVCE can be challenging for freshers and visitors alike. This digital map is your solution—a comprehensive **floor-by-floor guide** to the K.R. Circle campus, created by manually surveying every corridor, lab, and department building.

## How to Use the Map

**1. Open the Layer Menu** Click the square arrow icon positioned at the top-left corner of the map's title bar to open the side panel.

**2. Filter by Floor** Uncheck the boxes (e.g., "Ground Floor") to hide layers you do not need. This clears up the visual clutter and allows you to focus strictly on the floor you are trying to navigate.

## Map Legend: Decoding the Icons

To make navigation intuitive, we have assigned specific icons and colors to different types of rooms across the campus. Look for these indicators when exploring the map:

| Facility Type | Map Icon / Indicator |
| :--- | :--- |
| **Laboratories** | Wrench Icon |
| **Classrooms** | Amusement Park Icon |
| **Lecture Halls** | Graduation Cap Icon |
| **Staff Cabins** | Target Icon |
| **Administrative & Dept Offices** | Building / Pillar Icon |
| **Placement Office (TPO)** | Mountain Icon |
| **Restrooms** | Standard Gents & Ladies Icons |
| **Miscellaneous (Minchu, etc.)** | Alien Icon |
