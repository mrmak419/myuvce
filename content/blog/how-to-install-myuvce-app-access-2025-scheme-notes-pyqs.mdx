## 1. Install MyUVCE Hub (Attendance, SGPA, Directory & Notes)

The Hub is a utility application that calculates attendance margins, tracks SGPA based on the autonomous university regulations, and provides access to the campus directory and academic resources from the 2nd semester onwards.

### Android: Download from Play Store
The Hub is available on the Google Play Store for Android devices.

1. **[Download MyUVCE Hub from Google Play](https://play.google.com/store/apps/details?id=in.myuvce.hub)**
2. Tap **Install** to add the app to your device.

### iOS & Desktop: Install via Browser
iOS and desktop users can install the Hub as a Progressive Web App.

1. Open **hub.myuvce.in** in Safari (iOS) or Chrome (Desktop).
2. On iOS: Tap the **Share Icon** and select **Add to Home Screen**.
3. On Desktop/Android Chrome: Click the **Install App** prompt in the URL bar or browser menu.

## 2. Install the MyUVCE Website (1st Sem Notes & Articles)

<Callout type="info" title="Automated Installation Available">
  **Notice:** If you are browsing on Chrome (Android) or Edge, you may see an automated "Install MyUVCE App" prompt at the bottom of your screen. Click **Install App** to skip the manual steps below.
</Callout>

### Method 1: Android App (APK)
This method installs a dedicated app drawer shortcut for the main MyUVCE site.

1. **[Download MyUVCE APK](https://drive.google.com/uc?export=download&id=1woQwEmJ4ZeZGQa76dKLjVW7m1nFjER3q)**
2. Once downloaded, tap the file and select **Package Installer** to complete the setup.

*Security Note:* If Google Play Protect blocks the installation, tap **Install anyway**. This is an independent student project.

### Method 2: iOS / iPhone (Safari)
iPhones do not support APK files. Use Safari to add the PWA to your home screen.

1. Open **myuvce.in** in the Safari browser.
2. Tap the **Share Icon** (the square with an upward arrow at the bottom).
3. Scroll down the menu and select **Add to Home Screen**.
4. Tap **Add** in the top right corner.

### Method 3: Android (Browser Setup)
If you prefer not to install the APK, you can install the Progressive Web App directly from Chrome.

1. Open **myuvce.in** on Chrome and tap the **Three Dots (⋮)** menu.
2. Select **Add to Home Screen** (or "Install App").
3. Tap **Install** (Do not select "Create Shortcut" if prompted).
