## Complete Hackathon Roadmap

Follow this end-to-end roadmap to create winning hackathon projects using the latest AI tools and proven development strategies. Expand each phase below for specific tools and sample prompts.

<AccordionGroup>
  <Accordion title="Phase 1: Idea & Research">
    **Goal:** Find your idea, define the problem statement, and create a strong USP (Unique Selling Proposition).

    **AI Tools:**
    * **ChatGPT** ([chat.openai.com](https://chat.openai.com)): Idea refinement & brainstorming
    * **Claude / Gemini** ([gemini.google.com](https://gemini.google.com)): Alternative solutions & research

    **Sample Prompts:**
    * *"Suggest 5 unique hackathon project ideas related to [your domain]."*
    * *"Summarize the problem of [problem] in 2 lines for a hackathon pitch."*
    * *"Help me create a strong USP for [idea]."*
  </Accordion>

  <Accordion title="Phase 2: UI/UX Design">
    **Goal:** Create stunning wireframes, prototypes, and app designs that wow the judges.

    **Design Tools:**
    * **Stitch** ([stitch.withgoogle.com](https://developers.googleblog.com/en/stitch-a-new-way-to-design-uis/)): AI-powered UI generation
    * **Builder.io** ([builder.io](https://www.builder.io)): Drag-and-drop prototyping
    * **Vo.dev** ([vo.dev](https://vo.dev)): Low-code prototyping

    **Sample Prompts:**
    * *"Stitch, create a modern e-commerce app with dark-mode UI and minimal buttons."*
    * *"Builder.io, design a dashboard layout with a sidebar and analytics charts."*
  </Accordion>

  <Accordion title="Phase 3: Development & APIs">
    **Goal:** Build your backend, frontend, and integrate APIs rapidly.

    **Development Tools:**
    * **Replit AI** ([replit.com/ai](https://replit.com/ai)): Full-stack development
    * **Lovable** ([lovable.dev](https://lovable.dev)): Rapid prototyping
    * **Cursor AI** ([cursor.sh](https://cursor.sh)): Code debugging & refinement

    **Sample Prompts:**
    * *"Replit Agent, make me a weather app with a search bar and weather icons."*
    * *"Lovable, create an authentication system with signup/login and JWT."*
    * *"Cursor, optimize this API code for better performance and lower latency."*
  </Accordion>

  <Accordion title="Phase 4: Debugging & Polishing">
    **Goal:** Fix errors, test thoroughly, and optimize performance before the submission deadline.

    **Debugging Tools:**
    * **Intellify** ([intellify.io](https://intellify.io)): Smart debugging assistant
    * **Highlight** ([highlight.io](https://highlight.io)): Monitoring & bug tracking

    **Sample Prompts:**
    * *"Here is my error log: [paste log]. Suggest possible fixes."*
    * *"Generate edge-case test cases for my endpoint: /user/login"*
  </Accordion>

  <Accordion title="Phase 5: Image & Visual Assets">
    **Goal:** Create stunning, copyright-free visuals for your slides and presentations.

    **Image Generation Tools:**
    * **Fooocus** ([fooocus.one](https://fooocus.one/en)): High-quality AI image generation
    * **Google Colab + Stable Diffusion** ([colab.research.google.com](https://colab.research.google.com)): Advanced, localized image creation

    **Sample Prompts:**
    * *"Fooocus, generate a futuristic dashboard illustration with neon blue lights."*
    * *"Create a conceptual, minimalist image showing teamwork and coding collaboration."*
  </Accordion>

  <Accordion title="Phase 6: Presentation & Pitch">
    **Goal:** Create a winning pitch deck that clearly communicates your USP and impresses judges.

    **Presentation Templates:**
    * **[Figma Hackathon Deck](https://www.figma.com/community/file/1392890526109576151)**
    * **[SlidesGo Hackathon Proposal](https://slidesgo.com/theme/hackathon-project-proposal)**
    * **[SlidesCarnival Programming](https://www.slidescarnival.com/tag/programming)**
    * **[Canva Tech Templates](https://www.canva.com/presentations/templates/technology/)**

    **Sample Prompts:**
    * *"Write concise content for a slide detailing the Problem Statement of [project name]."*
    * *"Outline a 5-slide hackathon pitch flow covering Intro, Problem, Solution, Demo, and Future Scope."*
  </Accordion>
</AccordionGroup>

## Important Pro-Tips & Notes

* **Check Official Sites:** Always check official hackathon websites for the latest dates and eligibility criteria, as schedules frequently change.
* **Team Building:** Most hackathons require team formation—start networking with your peers early.
* **Tool Practice:** Practice using the AI tools mentioned in Phases 1-6 *before* the actual hackathon timer starts.
* **Digital Footprint:** Keep your GitHub profile updated with the projects you build. Top repositories for practice include the [Developer Roadmap](https://github.com/kamranahmedse/developer-roadmap) and [Public APIs Collection](https://github.com/public-apis/public-apis).
* **Stay Alert:** Follow target companies on LinkedIn and join platforms like Unstop, HackerEarth, and Devpost.
